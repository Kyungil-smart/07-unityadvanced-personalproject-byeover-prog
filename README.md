# 🎮 Game Controls

우리 게임은 **130 BPM 리듬에 맞춰 몬스터(노트)를 처리하는 리듬 액션 게임**입니다.  
플레이어는 자동으로 이동하며, **정확한 타이밍 입력이 핵심**입니다.

---

## ⌨ 기본 조작

| 키 | 기능 |
|---|---|
| **Space** | 공격 / 노트 타격 |
| **Esc** | 게임 일시정지 |
| **Enter** | 결과 화면에서 다음 진행 |

---

## 🎵 플레이 방법

1. 몬스터(노트)는 음악의 **박자에 맞춰 등장**합니다.  
2. 노트가 **Hit Line에 도달하는 순간 Space 키를 입력**하면 공격이 성공합니다.  
3. 타이밍이 맞으면 **Hit 판정**, 맞지 않으면 **Miss 판정**이 발생합니다.  
4. Miss가 발생하면 몬스터가 플레이어에게 접근하여 **체력을 감소**시킵니다.  
5. 성공적으로 노트를 처리하면 **콤보가 누적**됩니다.

---

## ❤️ 플레이어 체력

- 플레이어는 **하트 10칸 체력**을 가지고 있습니다.
- 몬스터가 플레이어에게 닿으면 **하트가 감소**합니다.
- 체력이 모두 소진되면 **게임 오버**가 됩니다.
